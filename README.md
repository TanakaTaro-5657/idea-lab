# idea-lab

idea-lab は、小さなゲーム案や UI 実験を気軽に置いて試すための実験場です。

## 最初のゲーム

`index.html` をブラウザで開くと、1画面だけの選択肢ノベルゲームを遊べます。
外部ライブラリは使わず、HTML/CSS/JavaScript だけで動くため、GitHub Pages でもそのまま表示できます。

## 動作確認方法

1. リポジトリをローカルに取得します。
2. `index.html` をブラウザで開きます。
3. 表示された選択肢をクリックして、文章が変わることを確認します。

ローカルサーバーで確認する場合は、次のコマンドを実行して `http://localhost:8000` にアクセスします。

```bash
python3 -m http.server 8000
```
